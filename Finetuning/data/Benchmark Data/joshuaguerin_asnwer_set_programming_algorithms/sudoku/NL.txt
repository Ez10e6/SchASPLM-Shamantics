You are tasked with solving a standard 9x9 Sudoku puzzle.
The grid is defined by rows and columns indexed 1 to 9.
The grid is also divided into nine 3x3 subgrids.

The rules are:
1. Each cell in the grid must contain exactly one number from 1 to 9.
2. Each row must contain exactly one occurrence of each number.
3. Each column must contain exactly one occurrence of each number.
4. Each 3x3 subgrid must contain exactly one occurrence of each number.

The output should be the filled grid.
